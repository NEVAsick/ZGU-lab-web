# Лабораторная работа 3: Разработка веб-приложения с использованием чистого JavaScript и Bootstrap

#### **Цель работы:** Освоение базовых инструментов фронтенд-разработки, включая Node.js, npm, Bootstrap 5 и нативные ES-модули. Реализация двухстраничного веб-приложения с динамической генерацией контента без использования фреймворков. В рамках работы применяются компонентный подход, манипуляции с DOM, обработка событий и система уведомлений.

## Функциональность

*   **Двухстраничное приложение:** главная страница (список элементов) и страница детального просмотра.
*   **Динамическая генерация карточек** на основе данных (без хардкода в HTML).
*   **Всплывающие уведомления (Toasts/Alerts)** — отображаются при действиях пользователя 
*   **Навигация:**
    *   Кнопка «Домой»/«Назад» для возврата на главную страницу.
*   **Сброс состояния при перезагрузке** — после обновления страницы отображаются только исходные карточки.
*   **Адаптивная вёрстка** с использованием Bootstrap 5.
*   **Чистая архитектура:** разделение на страницы и компоненты, инкапсуляция логики.

## Инструкция по запуску

**Требования:**

*   Установленный Node.js (для работы с npm).
*   Расширение Live Server в VS Code (или любой локальный веб-сервер).

**Шаги:**

1.  **Установите зависимости:**

    ```bash
    npm install
    ```

    Команда установит Bootstrap в папку `node_modules/`.



2.  **Запустите приложение:**
    *   Откройте проект в VS Code.
    *   Нажмите ПКМ на файле `index.html` → **Open with Live Server** (или используйте другой локальный сервер).
    *   Приложение откроется в браузере (обычно на `http://127.0.0.1:5500`).

**Важно:** Приложение не требует серверной части. Вся логика выполняется в браузере.
