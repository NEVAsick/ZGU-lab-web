# Лабораторная работа №2: TypeScript Калькулятор

## Цель работы
Реализация интерактивной логики калькулятора с использованием TypeScript, обработка событий и управление DOM-элементами.

## Реализованная функциональность

### Базовая логика
- Обработка цифрового ввода (0-9, точка)
- Основные операции: `+`, `-`, `×`, `÷`
- Вычисление результата (`=`)
- Очистка ввода (`C`)

### Дополнительные функции
1. Операция смены знака (`+/-`)
2. Операция вычисления процента (`%`)
3. Кнопка стирания введенной цифры назад (`←`)
4. Смена цвета фона по кнопке
5. Операция вычисления квадратного корня (`√`)
6. Операция возведения в квадрат (`x²`)
7. Операция вычисления факториала (`x!`)
8. Кнопка добавления трех нулей за раз (`000`)
9. Накапливаемое сложение
10. Накапливаемое вычитание
11. Смена цвета окна вывода результата по кнопке
12. Индивидуальная операция (индивидуально)

## Запуск проекта

### Способ 1: Live Server (рекомендуется)
1. Установите расширение **Live Server** в VS Code
2. Откройте файл `index.html`
3. Нажмите кнопку "Go Live" в правом нижнем углу VS Code

### Способ 2: Прямое открытие
1. Откройте файл `index.html` напрямую в браузере
   - Google Chrome
   - Mozilla Firefox
   - Microsoft Edge
   - Safari

### Требования
- Любой современный браузер с поддержкой ES6+
- Для разработки: TypeScript компилятор (tsc) установлен глобально или в проекте
- В проекте настроена автоматическая компиляция TypeScript в JavaScript

### Порядок работы с кодом
1. Исходный код на TypeScript находится в файле `calculator.ts`
2. При сохранении файла `calculator.ts` автоматически компилируется в `calculator.js`
3. HTML-файл подключает скомпилированный JavaScript файл
4. Все изменения в логике калькулятора вносятся в TypeScript файл
