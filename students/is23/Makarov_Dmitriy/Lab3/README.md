# Отчёт по лабораторной работе №3

---

## **Тема**

**Простое веб-приложение. Верстка**

---

## **Цель работы**

Цель данной лабораторной работы - знакомство с **node**, **npm**, написание простого приложения на **JavaScript**. В ходе выполнения работы, нам предстоит ознакомиться с кодом реализации простого интерфейса и вывода данных, и затем выполнить задания по варианту.

---

## **Задание**

1. Реализовать одностраничное веб-приложение (SPA).  
2. Использовать модульную структуру JavaScript (`import` / `export`).  
3. Реализовать компоненты интерфейса с помощью классов.  
4. Организовать навигацию между страницами без перезагрузки.  
5. Реализовать отображение списка элементов и страницы с подробной информацией.

---

## **Ход выполнения работы**

1. Создано SPA-приложение с точкой входа `main.js`.  
2. Реализована модульная структура проекта:
   - страницы (`pages`);
   - компоненты (`components`);
   - точка входа приложения.
3. Создана главная страница со списком элементов (каталог кошек).  
4. Для отображения элементов списка реализован компонент карточки.  
5. Реализована страница с подробной информацией об элементе.  
6. Навигация между страницами осуществляется без перезагрузки страницы путём динамической замены содержимого DOM.  
7. Использованы классы JavaScript для описания страниц и компонентов.  
8. Реализована обработка пользовательских событий (нажатие кнопок).

---

## **Результат**

Получено одностраничное веб-приложение с каталогом элементов, возможностью просмотра списка и перехода на страницу с подробной информацией без перезагрузки страницы.

---

## **Вывод**

В ходе лабораторной работы были изучены принципы построения SPA-приложений, работа с ES-модулями, классами и компонентным подходом. Получены навыки динамического управления DOM и организации структуры клиентского приложения.
