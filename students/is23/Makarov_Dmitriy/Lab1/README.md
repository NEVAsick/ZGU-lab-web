# Отчёт по лабораторной работе №1

---

## **Тема**

**Калькулятор на HTML/CSS**

---

## **Цель работы**

Цель данной лабораторной работы - знакомство с инструментами построения пользовательских интерфейсов web-сайтов: **HTML**, **CSS**. В ходе выполнения работы, нам предстоит ознакомиться с кодом реализации простого калькулятора, и затем выполнить задания по варианту.

---

## **Задание**

1. Сверстать страницу с калькулятором (кнопки, поле вывода).

2. Добавить блок с целью ЛР и выделить слова: знакомство, **HTML**, **CSS** (через тег для подсветки).

3. Сделать фон страницы картинкой по ссылке из интернета.

4. Добавить кнопку (ссылку) на GitHub-профиль.

5. Добавить надпись с автором и сделать её сворачиваемой/разворачиваемой.

---

## **Ход выполнения работы**

1. Создана HTML-разметка калькулятора: контейнер, поле вывода, сетка кнопок.

2. Оформление выполнено через **CSS**: выравнивание, размеры, отступы, стили кнопок.

3. Добавлен блок с целью лабораторной работы; ключевые слова выделены тегом подсветки.

4. Реализован фон страницы через background-image: url(...) (картинка по ссылке).

5. Добавлена кнопка-ссылка на GitHub (переход на http://github.com/LightBear124).

6. Блок “Автор” реализован с возможностью сворачивания/разворачивания (например, через <details><summary>).

---

## **Результат**

Получена веб-страница с интерфейсом калькулятора и дополнительными элементами: цель ЛР с подсветкой слов, фон-картинка по ссылке, кнопка на GitHub, сворачиваемый блок автора.

---

## **Вывод**

В ходе работы освоены базовые навыки разметки HTML-страницы и её оформления с помощью CSS, а также добавление ссылок и простых интерактивных элементов интерфейса средствами HTML.
