# Отчёт по лабораторной работе №6

---

## **Тема**

**Разработка серверной части приложения на Node.js**

---

## **Цель работы**

Цель работы - изучение основ разработки серверной части веб-приложения с использованием **Node.js** и **Express.js**. В рамках лабораторной работы нам надо создать полноценное **REST API** для галереи изображений, организовать структуру проекта по принципу разделения ответственности (Controller-Service-DAO-Repository) и интегрировать созданный бэкенд с клиентской частью приложения из предыдущих лабораторных работ.

---

## **Задание**

1. Разработать серверное приложение на Node.js с использованием Express.  
2. Реализовать REST API для получения данных.  
3. Организовать хранение данных в формате JSON.  
4. Реализовать архитектуру с разделением на слои (Controller, Service, DAO, Repository).  
5. Обеспечить взаимодействие клиентского приложения с сервером через Fetch API.  

---

## **Ход выполнения работы**

1. Создан сервер на платформе **Node.js** с использованием фреймворка **Express**.  
2. Реализована раздача статических файлов клиентского приложения.  
3. Данные вынесены из клиентской части и хранятся в JSON-файле на сервере.  
4. Реализованы REST-эндпоинты:
   - `GET /cats` — получение списка элементов;
   - `GET /cats/:id` — получение элемента по идентификатору.
5. Реализована многоуровневая архитектура backend-приложения:
   - **Controller** — обработка HTTP-запросов и формирование ответов;
   - **Service** — бизнес-логика и валидация данных;
   - **DAO / Repository** — работа с источником данных;
   - **DBConnector** — чтение и запись данных в файл.
6. Клиентская часть приложения получает данные с сервера с использованием **Fetch API**.  
7. Взаимодействие клиента и сервера осуществляется в рамках одного домена, что исключает проблемы с CORS.

---

## **Результат**

Реализовано клиент-серверное веб-приложение, в котором данные загружаются с backend-сервера через REST API. Клиентское приложение динамически отображает данные без перезагрузки страницы.

---

## **Вывод**

В ходе лабораторной работы были изучены основы разработки серверных приложений на Node.js, принципы REST API и клиент-серверного взаимодействия. Полученные навыки позволяют создавать полнофункциональные веб-приложения с разделением frontend и backend логики.
