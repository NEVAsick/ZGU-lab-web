# Отчёт по лабораторной работе №2

---

## **Тема**

**Калькулятор на JavaScript**

---

## **Цель работы**

Цель данной лабораторной работы - знакомство с инструментами построения пользовательских интерфейсов web-сайтов: **HTML**, **CSS**, **JavaScript**. В ходе выполнения работы, нам предстоит продолжить реализовывать простой калькулятор, и затем выполнить задания по варианту.

---

## **Задание**

1. Реализовать логику калькулятора с использованием JavaScript / TypeScript.  
2. Освоить базовые типы данных и массивы.  
3. Использовать функции и обработчики событий.  
4. Реализовать дополнительные операции калькулятора.  
5. Ознакомиться с дженериками и встроенными типами TypeScript.

---

## **Ход выполнения работы**

1. Реализована логика калькулятора с обработкой нажатий кнопок.  
2. Использованы базовые типы данных TypeScript: `number`, `string`, `boolean`.  
3. Реализована работа с массивами чисел и строк, а также массивами смешанных типов.  
4. Добавлены дополнительные операции калькулятора:
   - смена знака числа (**+/−**);
   - вычисление процента (**%**);
   - удаление последнего символа (**backspace**);
   - вычисление квадратного корня;
   - возведение числа в квадрат;
   - вычисление факториала;
   - добавление сразу трёх нулей (**000**).
5. Использованы функции и обработчики событий для взаимодействия с пользовательским интерфейсом.  
6. Ознакомление с дженериками на примерах универсальных типов:
   - `ApiResponse<T>`;
   - `Partial<T>`.

---

## **Результат**

Создан функциональный калькулятор с расширенным набором операций. Реализована логика вычислений с использованием JavaScript и TypeScript, а также применена статическая типизация и дженерики.

---

## **Вывод**

В ходе лабораторной работы были изучены основы JavaScript и TypeScript, работа с типами данных, массивами и функциями. Получены навыки реализации логики приложения и использования возможностей типизации TypeScript для повышения надёжности кода.
