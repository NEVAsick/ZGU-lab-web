# Отчёт по лабораторной работе №4

---

## **Тема**

**РПростое веб-приложение. Работа с API**

---

## **Цель работы**

Цель данной лабораторной работы - взаимодействие с внешним Api через **XMLHttpRequest**. В ходе выполнения работы, нам предстоит ознакомиться с кодом реализации простого взаимодействия с внешним Api, получение данных и вывод их в интерфейс пользователя, и затем выполнить задания по варианту.

---

## **Задание**

1. Реализовать веб-приложение для получения данных через VK API.  
2. Получить список участников сообщества.  
3. Реализовать фильтрацию участников по заданному параметру (вариант 2).  
4. Обработать возможные ошибки при выполнении запросов.  
5. Отобразить результат на странице без перезагрузки.

---

## **Ход выполнения работы**

1. Создано веб-приложение с использованием HTML, CSS и JavaScript.  
2. Реализован HTTP-запрос к VK API с использованием `fetch`.  
3. Использованы асинхронные функции (`async/await`) для работы с ответами сервера.  
4. Полученные данные обработаны и выведены в пользовательский интерфейс.  
5. Реализован фильтр участников на основе параметров API.  
6. Добавлена обработка ошибок сети и ошибок ответа API с выводом сообщений пользователю.  
7. Приложение запускается через локальный сервер (Live Server), что позволяет корректно выполнять запросы.

---

## **Результат**

Создано клиентское веб-приложение, которое получает данные из VK API, отображает список участников сообщества и позволяет выполнять фильтрацию данных без перезагрузки страницы.

---

## **Вывод**

В ходе лабораторной работы были изучены основы работы с внешними API, асинхронные HTTP-запросы, обработка ошибок и динамическое обновление интерфейса. Полученные знания позволяют создавать веб-приложения, взаимодействующие с серверными сервисами.
